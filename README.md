#### java cli
+ app de ejemplo `HolaMundo.java`
+ `javac HolaMundo.java`
+ resulta en `HolaMundo.class`
+ `java HolaMundo`
+ `vi HolaMundo.class`
  + para ver hex `:%!xxd` para salir `:%!xxd -r` o `:q!`
+ herramienta de conversion de java `javap -c HolaMundo.class`
